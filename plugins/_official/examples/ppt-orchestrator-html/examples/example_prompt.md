# Example Prompt

## 草稿 PPT 转 HTML 型 PPT

基于我上传的草稿 PPT，生成 HTML 型可编辑 PPT。

要求：

1. 保持所有正式信息不改动、不缺失。
2. 明显编辑指令 / 待修改占位符不作为最终页面正文展示。
3. 先建立内容账本，再判断页面类型。
4. 页面类型从总控 Skill 的页面类型池中选择。
5. 默认绑定超宽大屏商务增长风格，尺寸为 3696×1008。
6. 如果内容过多，优先拆页，禁止缩小字号硬塞或删除信息。
7. 输出页面规划 JSON、HTML 代码和内容完整性检查结果。

## 单页测试 Prompt

请只测试第 1 页：

- 输出 content_inventory
- 输出 edit_instruction_inventory
- 判断页面类型
- 输出 page_plan
- 暂不生成最终 HTML

重点检查页面类型是否正确、正式内容是否完整、编辑备注是否被排除。
