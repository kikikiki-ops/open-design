# Style Binding

## 1. 绑定时机

只有 `structureFrozen = true` 后，才允许调用视觉风格 Skill。

## 2. 分工

总控固定：页面角色、页面类型、布局组件、内容数量、内容映射、拆页结果。

视觉风格负责：背景、色彩、字体、字号层级、Logo、线条、卡片、图标、图表和装饰。

## 3. 参考图隔离

视觉参考图默认只提取：

- 色彩；
- 字体气质；
- 背景与装饰；
- Logo、线条、圆角、图表样式；
- 同类页面的视觉表达。

默认不提取：

- 对所有页面通用的版式；
- 页面类型；
- 内容删减策略。

只有 `semanticPattern`、内容容量和页面角色均匹配时，才可复用参考页布局。

## 4. Logo

Logo 规范由视觉风格 Skill 定义，总控只输出：

```json
{"showLogo": true, "logoVariant": "auto"}
```

Logo 必须是独立组件，不得烘焙在背景图中。
