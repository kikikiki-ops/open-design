# Edit Instruction & Placeholder Rules

## 1. 识别对象

占位符：`xxx`、`XX`、`TBD`、`待补充`、`+pp`、未完成单位、临时代称。

编辑指令：`改成...`、`替换为...`、`删除...`、`调整为...`、`增加...`、`不要...` 等内部修改说明。

## 2. 分类

- 明确占位符 → `do-not-render`，进入 `edit_instruction_inventory`；
- 明确编辑指令且替换边界清晰 → 应用目标表达，并记录应用日志；
- 正文混有占位片段 → 正式部分保留，占位部分不渲染；
- 无法判断 → `pending-confirmation`，不得直接美化为正式正文。

## 3. 禁止

- 把编辑备注当作演示文案；
- 不留审校记录地直接删除；
- 根据模糊备注擅自重写业务结论。
