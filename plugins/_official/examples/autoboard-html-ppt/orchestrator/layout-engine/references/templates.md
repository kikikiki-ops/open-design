# 页面模板详细规范

画布比例 3.67 : 1，所有尺寸为内容区域百分比。

---

## T01：目录 / 章节导航页

**适用场景**：报告目录 / 会议议程 / 战略章节 / 阶段结构

**页面区域**

| 区域 | 高度比例 |
|------|----------|
| header | 15% |
| main | 73% |
| footer | 12% |

**main 内部列分配**

| 章节数 | 列比例 |
|--------|--------|
| 2 | 50:50 |
| 3 | 33:33:33 |
| 4 | 25:25:25:25 |
| 5+ | 两行布局或分组 |

**单章节模块结构**

```text
编号 → 章节标题 → 章节说明
```

---

## T02：核心数据展示页

**适用场景**：多项业绩 / 增长指标 / 业务成果 / 核心方向

**页面区域**

| 区域 | 高度比例 |
|------|----------|
| header | 10% |
| main | 80% |
| footer | 10% |

**列宽自适应**

| 指标数量 | 列比例 |
|----------|--------|
| 2 | 50:50 |
| 3 | 33:33:33 |
| 4 | 25×4 |
| 5–6 | 2×3 网格 |
| 7+ | 拆页 |

**单卡片结构**

```text
分类标签 → 指标名称 → 核心数据 → 图表 → 结论说明
```

**图表匹配**

| 数据关系 | 推荐图表 |
|----------|----------|
| 增长趋势 | 折线图 |
| 前后比较 | 柱状图 |
| 占比 | 环形图 |
| 资源分配 | 饼图 |
| 阶段变化 | 阶梯图 |
| 多指标变化 | 组合图 |

## T03：中心概念 / 双边连接页

**适用场景**：平台连接供需 / 广告主与媒体 / 输入与输出 / 生态协同

**页面区域**

| 区域 | 高度比例 |
|------|----------|
| header | 10% |
| main | 80% |
| footer | 10% |

**main 列分配**：25:50:25（左:中:右）

**角色模块**

```text
角色标题 → Lucide 图标 → 当前状态 → 未来变化 → 说明卡片
```

**自适应规则**
- 中央概念是页面唯一视觉核心
- 左右视觉重量尽量接近
- 关系线超过 5 条时改为矩阵或流程图
- 关系线不能穿过核心文字

---

## T04：左右策略对比页

**适用场景**：过去与现在 / 旧方案与新方案 / 优化前后

**页面区域**

| 区域 | 高度比例 |
|------|----------|
| header | 10% |
| main | 80% |
| footer | 10% |

**main 列分配**：50:50

**单侧模块**

```text
策略名称 → 策略说明 → 关键机制 → 图示/图表 → 最终结果
```

**对比规则**
- 左右标题使用相同句式
- 对比维度必须一致
- 对比结果位于相同水平位置
- 对比项超过 4 个时改为对比表或矩阵

**适用场景**：业务机制 / 算法模型 / 定价策略 / 投放优化

| 区域 | 高度比例 |
|------|----------|
| header | 10% |
| main | 85% |
| footer | 5% |

**main 列分配**：17:42:25:17

**各模块内部结构**

```text

调节：目标 → 方法 → 调节前 → 调节后 → 趋势

```

## T07：曲线路径 / 战略路线图

**适用场景**：长期战略 / 技术演进 / 增长曲线

**页面区域**

| 区域 | 高度比例 |
|------|----------|
| header | 10% |
| main | 85% |
| footer | 5% |

**main**：路径区域占 100%，内含 3–6 个节点

**节点结构**

```text
Lucide 图标（可选） → 节点名称 → 节点说明 → 节点价值 → 关联能力
```

**路径规则**
- 路径线有方向，默认从左向右
- 节点不互相遮挡
- 路径线不穿过核心文字
