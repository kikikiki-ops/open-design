# 质量检查清单
# 超宽大屏商务增长风格

## 1. 画布检查

```text
[ ] 画布是否为 3696 × 1008 px
[ ] 是否适配 11m × 3m 超宽大屏
[ ] 是否是 11:3 超宽构图，而不是 16:9 横向拉伸
[ ] 是否无滚动、无裁切、无内容溢出
```

## 2. 安全区检查

```text
[ ] 核心信息是否位于 X: 220~3476 px 范围内
[ ] 核心信息是否位于 Y: 90~918 px 范围内
[ ] 左右边缘是否没有重要文字
[ ] 顶部和底部是否预留了显示安全空间
```

## 3. 版式检查

```text
[ ] 页面是否中心聚焦
[ ] 左右两侧是否主要用于氛围延展
[ ] 主体模块是否横向展开
[ ] 页面是否没有被平均铺满
[ ] 模块之间间距是否均匀
[ ] 视觉重心是否清晰
```

## 4. 字体检查

```text
[ ] 主标题是否 60~72 px
[ ] 副标题是否 24~30 px
[ ] 正文字号是否不小于 16 px
[ ] 关键数字是否 72~88 px
[ ] 标题是否不超过 2 行
[ ] 是否没有使用花哨字体 / 手写体 / 卡通体
```

## 5. 色彩检查

```text
[ ] 背景是否为浅暖白 / 米白
[ ] 主文字是否为深青黑
[ ] 重点结果是否使用香槟金
[ ] 策略 / 数据 / 智能信息是否使用青绿色
[ ] 是否没有出现赛博蓝紫、高饱和霓虹色
```

## 6. 卡片检查

```text
[ ] 主体是否使用白色大圆角卡片
[ ] 卡片圆角是否约 22 px
[ ] 卡片边框是否极细
[ ] 卡片阴影是否极轻
[ ] 卡片内边距是否充足
[ ] 是否没有厚重投影和拟物按钮感
```

## 7. 背景检查

```text
[ ] 是否有低透明青绿色 + 香槟金柔和流线
[ ] 流线是否位于底部或左右侧
[ ] 背景是否不干扰正文
[ ] 是否没有强发光、复杂粒子、HUD 科技线框
```

## 8. 内容密度检查

```text
[ ] 是否一页只讲一个核心结论
[ ] 是否不超过 2~4 个主要模块
[ ] 是否不超过 2~4 个关键数字
[ ] 是否没有大段正文
[ ] 是否没有密集表格
[ ] 远距离是否一眼可读
```

## 9. 风格一致性检查

```text
[ ] 是否高级、明亮、克制、商务
[ ] 是否具备轻科技感但不过度炫光
[ ] 是否保留品牌营销 / 生意增长气质
[ ] 是否避免模板感、卡通感、脏乱感
```


---

## 背景图检查

```text
[ ] 是否按照页面角色使用 cover / content / closing 背景
[ ] 背景图是否适配 3696 × 1008
[ ] 背景图是否只承载氛围，不承载必须编辑的信息
[ ] 主标题、正文、数字、Logo、二维码是否仍为独立可编辑元素
[ ] 内容页背景是否比封面背景更克制，不干扰阅读
```

## 固定品牌信息检查

```text
[ ] 是否没有额外生成第二个“快手联盟”文本
[ ] 是否没有叠加重复 Logo
[ ] 是否没有把顶部品牌区自动生成为常驻文字
[ ] 会议名、年份、演讲人等是否只在原始内容提供时展示
[ ] 背景图自带标识与 HTML 可编辑内容是否没有重复
```

## 独立 Logo 组件检查

```text
[ ] 背景图是否无 Logo
[ ] Logo 是否通过 FixedBrandLogo 独立组件渲染
[ ] 封面 / 内容 / 封尾是否都显示 Logo
[ ] Logo 是否使用 SVG 或高清透明 PNG
[ ] 是否没有重复生成“快手联盟”文字
[ ] 是否没有在 Logo 下方额外生成品牌副标题
[ ] Logo 是否未被背景图压缩导致模糊
```
