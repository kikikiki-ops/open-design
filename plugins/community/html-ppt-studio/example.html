Why open

The proof is real. The access isn't.

In April 2026, Claude Design proved an LLM can truly design — real artifacts, not essays about design. But it is closed-source, paid, and cloud-only.

  • Locked to a single model vendor
  • No agent choice, no self-hosting
  • No BYOK, no local files
  • Open Design sets that capability free

Workflow

Idea to delivery in four moves

Start

Describe your goal in one sentence, or pick a template or plugin.

Direct

Set the visual direction — palette, type, and spacing flow into generation.

Generate

The agent reads full context and produces real runnable files in a sandbox.

Ship

Hand off to engineering, or film it with HyperFrames. Memory compounds.

Process-steps layout: every choice you make becomes context for the next run — the platform learns your taste.
