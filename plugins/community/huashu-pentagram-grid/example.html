Section 04 / 工作流 Idea → Delivery
Workflow
从想法到交付。
01
选择起点
一句话描述目标，或从模板 / Plugin 直接选一个起点。
217+ 个插件可作起点
02
确定视觉方向
选定方向后，色板、字体、间距自动带入生成流程。
129 套设计系统
03
生成 Artifact
Agent 读取全部上下文，产出真实可运行的文件，沙盒内即时预览和修改。
21 个 Coding Agent
04
交付与沉淀
导出给工程继续开发，或用 HyperFrames 转为营销视频；每次选择都沉淀为下次的上下文。
越用越准 设计记忆沉淀
