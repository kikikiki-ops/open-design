# 超宽大屏商务增长风格 PPT Skill

这是一个用于 AutoBoard / OpenDesign PPT 生成的风格模板 Skill 包。

## 文件结构

```text
ultrawide_business_growth_ppt_skill/
├── SKILL.md              # Skill 主入口与执行规则
├── style.md              # 完整样式系统
├── components.md         # 页面组件与内容组件规范
├── checklist.md          # 生成质量检查清单
├── examples/
│   └── example_prompt.md # 示例提示词
└── references/           # 可放置后续参考图 / 样稿说明
```

## 固定规格

- 物理尺寸：11m × 3m
- 像素分辨率：3696 × 1008 px
- 比例：11:3

## 适用方向

- 会场主屏
- 品牌营销大会
- 商业化增长汇报
- 案例复盘
- 数据策略发布
- 超宽横屏 PPT / KV

## 核心风格

浅暖底、柔光流线、香槟金商务感、青绿色轻科技感、白色圆角卡片、中心聚焦、低信息密度。
